This project area focuses on scientific machine learning for reduced-order modeling and many-query engineering simulation.

The work includes neural-network surrogates, POD-NN models, autoencoder-based latent models, ML-based coefficient prediction, and data-driven surrogate models for parametric PDE systems.

**Keywords:** scientific machine learning, neural networks, autoencoders, surrogate modeling, parametric PDEs, reduced-order modeling, real-time simulation.
