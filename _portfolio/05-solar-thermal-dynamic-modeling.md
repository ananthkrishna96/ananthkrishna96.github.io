This research internship project developed physics-based solar-thermal models within the SunPeek D-CAT framework.

The work included collector-field dynamics, 1D pipe heat-loss modeling, lumped 0D heat-exchanger modeling, coupled forward simulations, validation and yield analysis using real operational data, sensor mapping, automated tests, and diagnostic plots.

**Keywords:** scientific computing, ODE models, solar-thermal systems, dynamic simulation, validation, Python, physics-based modeling.
