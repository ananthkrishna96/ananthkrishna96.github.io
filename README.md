# Anantha Krishnan Orunnukaran Mani - Academic Website

This repository contains the GitHub Pages academic website for Anantha Krishnan Orunnukaran Mani.

Website URL: https://ananthkrishna96.github.io/

Built using the Academic Pages / Minimal Mistakes Jekyll template.

## Main editable files

- `_config.yml` - site identity, sidebar profile, and links
- `_data/navigation.yml` - top navigation menu
- `_pages/about.md` - homepage
- `_pages/research.md` - research page
- `_pages/projects.md` - projects page
- `_pages/cv.md` - CV page
- `_pages/experience.md` - experience page
- `_pages/software.md` - software and technical skills page
- `_pages/contact.md` - contact page
- `files/WEBSITE_ANANTH_CV_2026.pdf` - public CV PDF
- `images/anantha_profile.jpg` - profile photo

## Deploy

Push to the `master` branch of `git@github.com:ananthkrishna96/ananthkrishna96.github.io.git`. GitHub Pages should deploy from `master` and `/root`.
